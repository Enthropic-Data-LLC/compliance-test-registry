# API Q1 / Q2 — Oil and Gas Quality Management Systems

**Registry version:** 2026.05
**Last updated:** 2026-05-20
**Scope:** Quality management systems for organizations manufacturing, remanufacturing, and repairing products used in the oil and gas industry (API Q1); quality management for wellsite service and intervention operations (API Q2)
**Authority:** American Petroleum Institute (API); certification through API-licensed third-party auditors
**Enforcing context:** Contractual requirement from oil and gas operators and service companies; required for products listed in API monogram program; ISO 9001:2015 is the foundation with API-specific additions
**Current edition:** API Q1 9th Edition (2013, reaffirmed); API Q2 1st Edition (2015)

## API Q1 vs Q2 scope distinction

| Standard | Applies to | Primary focus |
|---|---|---|
| API Q1 | Manufacturing organizations producing oil and gas equipment | Product realization; design; manufacturing processes; API Monogram |
| API Q2 | Service and intervention operations (workover, pressure pumping, completion) | Service delivery; critical service tools; wellsite operations |

## API Q1 — key supplements to ISO 9001 (9th Edition)

### Design and Development (stronger than ISO 9001)

| Requirement | Confidence | Notes |
|---|---|---|
| Design and development plan | DETERMINISTIC | Written plan with stages, reviews, responsibilities |
| Design input record | DETERMINISTIC | All design inputs documented (functional, performance, regulatory) |
| Design output record | DETERMINISTIC | Documents verifying outputs meet inputs |
| Design review records | DETERMINISTIC | Formal staged reviews; attendees; action items |
| Design verification record | DETERMINISTIC | Evidence that design outputs meet design inputs |
| Design validation record | DETERMINISTIC | Evidence that product meets intended use under intended conditions |
| Design transfer | PARAMETERIZED | Process for transferring approved design to production |
| Design change control | DETERMINISTIC | All changes documented, reviewed, and re-validated |

**API Q1 design records are more prescriptive than ISO 9001** — each of the above must be separately documented as distinct records.

### Contingency Planning (API-specific)

| Requirement | Confidence | Notes |

| Requirement | Confidence |
|---|---|

| Third-party audit (initial + triennial surveillance) | DETERMINISTIC |
