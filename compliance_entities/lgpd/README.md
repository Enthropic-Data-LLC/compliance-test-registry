# LGPD — Lei Geral de Proteção de Dados (Brazil)

**Authority:** Autoridade Nacional de Proteção de Dados (ANPD)
**Scope:** Processing of personal data of individuals in Brazil; applies to any organization (Brazilian or foreign) processing data in Brazil, offering goods/services to Brazilian individuals, or processing data collected in Brazil
