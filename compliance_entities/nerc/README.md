# NERC Reliability Standards Registry

**Authority:** North American Electric Reliability Corporation (NERC); enforced by Regional Entities (RE) under FERC oversight (U.S.) and NEB oversight (Canada)
**Scope:** Bulk Electric System (BES) owners, operators, and users across North America

## Subdirectories

| Directory | Coverage | Parse depth |
|---|---|---|
| [`ops/`](./ops/) | Non-CIP reliability standards: BAL, COM, EOP, FAC, INT, IRO, MOD, NUC, PER, PRC, TOP, TPL, VAR | Deep — 5 spec files covering PRC-005, FAC-003/008, EOP-004, COM-002, TOP-001, PER-003/005, BAL-003, MOD-025, TPL-001, EOP-005/008 |
| [`cip/`](./cip/) | CIP-002 through CIP-015 (cybersecurity standards) | Parsed — CIP-002, CIP-003, CIP-004, CIP-005, CIP-006, CIP-007 |
