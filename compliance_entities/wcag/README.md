# WCAG 2.2 / Section 508 — Digital Accessibility

**Authority:** W3C Web Accessibility Initiative (WAI) (WCAG); U.S. Access Board (Section 508); DOJ (ADA enforcement)
**Scope:** WCAG 2.2: international standard for accessibility of web content and software interfaces for people with disabilities (4 principles: Perceivable, Operable, Understandable, Robust; conformance levels A, AA, AAA) | Section 508: U.S. federal agencies and vendors; references WCAG 2.0 Level AA; ADA Title III enforcement extends accessibility obligations to commercial websites
