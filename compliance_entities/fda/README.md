# FDA Regulations Registry

**Authority:** U.S. Food and Drug Administration (FDA)

This directory contains RDF registries for FDA regulations across multiple program areas.

## Subdirectories

| Directory | Regulation | Scope | Parse depth |
|---|---|---|---|
| [`21cfr11/`](./21cfr11/) | 21 CFR Part 11 | Electronic records and electronic signatures for regulated industries | Deep — 2 spec files |
| [`21cfr210-211/`](./21cfr210-211/) | 21 CFR Parts 210/211 | Current Good Manufacturing Practice (cGMP) for drug manufacturing | Index only |
| [`fsma/`](./fsma/) | FSMA (21 USC 2201) | Food Safety Modernization Act — preventive controls for food | Index only |
| [`qmsr/`](./qmsr/) | 21 CFR Part 820 (QMSR) | Quality Management System Regulation for medical devices (aligns with ISO 13485) | Index only |
