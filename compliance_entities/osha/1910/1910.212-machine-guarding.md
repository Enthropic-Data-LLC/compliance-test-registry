# 1910.212 — General Requirements for All Machines (Machine Guarding)

## Scope summary

1910.212 establishes general machine guarding requirements for all general industry machinery. It is a performance-based standard: it requires guards at points of operation and other hazardous areas but does not prescribe specific guard designs (those are in subsequent sections: 1910.213 for woodworking, 1910.215 for abrasive wheels, 1910.216 for mills/calenders, 1910.217 for mechanical power presses, etc.). The general rule is the starting point; specific machinery standards take precedence where they exist.

The contested element is that OSHA's enforcement posture on "other hazardous parts" (power transmission components — belts, pulleys, gears, shafts, etc.) is highly inspector-dependent and has generated conflicting citation interpretations.

## (a)(1) — General Guarding Requirement

### Source excerpt

> *One or more methods of machine guarding shall be provided to protect the operator and other employees in the machine area from hazards such as those created by point of operation, ingoing nip points, rotating parts, flying chips and sparks.*

**Contested issue:** "Hazards such as those created by point of operation" — OSHA interprets this broadly. The boundary between what constitutes a "hazardous part" requiring guarding and normal machine operation varies between Regional Offices and inspection history. Industry relies on ANSI B11 standards for guidance, but compliance with ANSI is not explicitly required.

"""
1910.212(a)(1) — General Machine Guarding
Confidence: LOW (CONTESTED) | Human Review: MANDATORY
"""
import pytest
from datetime import datetime, timedelta


@pytest.mark.human_review_required
@pytest.mark.contested(
    regulation="1910.212(a)(1)",
    reason=(
        "Scope of 'hazardous parts' requiring guarding is interpreter-dependent; "
        "no bright-line test for what constitutes a guarding-required hazard"
    ),
)
def test_machine_guarding_human_determination_exists_and_current(
    machine_inventory, machine_guarding_assessments
):
    """
    Does not determine whether guarding is adequate.
    Verifies that a qualified human assessment exists and is current.
    """
    assessed_machine_ids = {a["machine_id"] for a in machine_guarding_assessments}
    unassessed = [
        m for m in machine_inventory
        if m["machine_id"] not in assessed_machine_ids
        and m.get("status") == "active"
    ]
    assert not unassessed, (
        f"UNRESOLVED: {len(unassessed)} active machine(s) have no guarding assessment: "
        f"{[m['machine_id'] for m in unassessed]}. "
        f"A qualified person must assess each machine for guarding requirements."
    )


@pytest.mark.human_review_required
@pytest.mark.contested(regulation="1910.212(a)(1)", reason="Guarding adequacy requires qualified-person determination")
def test_machine_guarding_assessments_current(machine_guarding_assessments):
    """Guarding assessments must not be stale."""
    from datetime import datetime, timedelta
    STALE_DAYS = 365
    stale = []
    for assessment in machine_guarding_assessments:
        age = datetime.utcnow().date() - assessment["assessment_date"]
        if age.days > STALE_DAYS:
            stale.append(
                f"Machine {assessment['machine_id']}: assessed {assessment['assessment_date']} "
                f"({age.days} days ago)"
            )
    assert not stale, (
        f"STALE: {len(stale)} machine guarding assessment(s) require re-review:\n"
        + "\n".join(stale)
    )


@pytest.mark.human_review_required
@pytest.mark.contested(regulation="1910.212(a)(1)")
def test_machine_guarding_assessments_have_qualified_assessor(machine_guarding_assessments):
    missing_assessor = [
        a for a in machine_guarding_assessments
        if not a.get("assessed_by") or not a.get("assessor_qualified")
    ]
    assert not missing_assessor, (
        f"INCOMPLETE: {len(missing_assessor)} guarding assessment(s) lack a qualified "
        f"assessor: {[a.get('machine_id') for a in missing_assessor]}"
    )

## Notes for the registry

- **ANSI B11 series:** While not mandatory, ANSI B11 standards (B11.0 General Safety Requirements, B11.1 through B11.26 for specific machine types) represent RAGAGEP for machine guarding. Documenting ANSI compliance alongside OSHA compliance strengthens the audit posture and resolves many contested-classification questions.
- **Point of operation vs. power transmission:** Guards at the point of operation (where work is performed — tooling, blades, dies) are clearest. Power transmission guards (belts, chains, gears, shafts) are governed by 1910.219 but 1910.212 is frequently cited as a catch-all when 1910.219 doesn't specifically address a component.
- **Guard removal procedures:** When a guard must be removed for maintenance, the LOTO program (1910.147) must be applied. The machine guarding assessment should note any guards that can be removed for service and confirm they are covered by the LOTO energy control procedure.
- **Inspection frequency:** (a)(1) does not specify an inspection interval. OSHA has cited employers for guards found to be missing during inspections where no maintenance records show the guard was deliberately removed. Annual guarding inspections are the industry norm.
