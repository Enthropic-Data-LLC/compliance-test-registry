## Scope summary

1910.147 establishes minimum performance requirements for controlling hazardous energy (electrical, mechanical, hydraulic, pneumatic, chemical, thermal, gravity) during servicing and maintenance of machinery and equipment. It applies when unexpected energization, startup, or release of stored energy could cause injury. The standard distinguishes between **authorized employees** (perform LOTO), **affected employees** (operate equipment subject to LOTO), and **other employees** (work in areas where LOTO is used).

Normal production operations are excluded unless: (1) employee must remove a guard or other safety device, or (2) employee must place any part of their body into the point of operation or area where work is performed during machine operation.

from datetime import datetime, timedelta

PROGRAM_REVIEW_INTERVAL_DAYS = 365


def test_loto_written_program_exists(written_programs):
    prog = next(
        (p for p in written_programs if p["standard_id"] == "1910.147"), None
    )
    assert prog is not None, (
        "VIOLATION: No written energy control (LOTO) program found for 1910.147."
    )


def test_loto_written_program_current(written_programs):
    prog = next(p for p in written_programs if p["standard_id"] == "1910.147")
    age = datetime.utcnow().date() - prog["last_reviewed_date"]
    assert age.days <= PROGRAM_REVIEW_INTERVAL_DAYS, (
        f"VIOLATION: LOTO written program is {age.days} days old. "
        f"Review and update required within {PROGRAM_REVIEW_INTERVAL_DAYS} days."
    )


def test_loto_written_program_covers_required_elements(written_programs):
    prog = next(p for p in written_programs if p["standard_id"] == "1910.147")
    required = {"procedures", "training", "periodic_inspections"}
    covered = set(prog.get("sections_covered", []))
    missing = required - covered
    assert not missing, (
        f"VIOLATION: LOTO program missing required elements: {missing}."
    )

REQUIRED_PROCEDURE_FIELDS = {

    "purpose",
    "authorization_rules",
    "energy_types",
    "isolation_points",
    "lockout_hardware",
    "de_energization_verification_steps",

def test_loto_procedure_exists_for_all_equipment(equipment, energy_control_procedures):
    proc_equipment_ids = {p["equipment_id"] for p in energy_control_procedures}

        f"VIOLATION: {len(covered_equipment)} equipment item(s) requiring LOTO "

"""
1910.147(c)(4) — Periodic (Annual) Inspection
Confidence: MEDIUM | Human Review: ASSUMPTION REQUIRED (ASSUME-147-001)
"""
import pytest
from datetime import datetime, timedelta

INSPECTION_INTERVAL_DAYS = 365


@pytest.mark.assumption(
    id="ASSUME-147-001",
    description="Adequate inspection certification = signed/dated record with corrective actions",
    approved_by="Compliance Officer",
    review_date="2026-05-20",
)
def test_loto_annual_inspection_current(energy_control_procedures, loto_inspection_records):
    violations = []
    for proc in energy_control_procedures:
        inspections = [
            r for r in loto_inspection_records
            if r["procedure_id"] == proc["procedure_id"]
        ]
        if not inspections:
            violations.append(
                f"Procedure {proc['procedure_id']}: no inspection record"
            )
            continue
        most_recent = max(inspections, key=lambda r: r["inspection_date"])
        age = datetime.utcnow().date() - most_recent["inspection_date"]
        if age.days > INSPECTION_INTERVAL_DAYS:
            violations.append(
                f"Procedure {proc['procedure_id']}: last inspection "
                f"{most_recent['inspection_date']} ({age.days} days ago)"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} LOTO procedure(s) past annual inspection interval:\n"
        + "\n".join(violations)
    )


@pytest.mark.assumption(id="ASSUME-147-001")
def test_loto_inspection_performed_by_independent_authorized_employee(
    loto_inspection_records, energy_control_procedures
):
    violations = []
    for record in loto_inspection_records:
        proc = next(
            (p for p in energy_control_procedures
             if p["procedure_id"] == record["procedure_id"]),
            None,
        )
        if proc and record["performed_by"] in proc.get("typical_users", []):
            violations.append(
                f"Inspection {record['inspection_id']}: performed by "
                f"{record['performed_by']} who is also a typical user "
                f"of procedure {record['procedure_id']}."
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} inspection(s) performed by the procedure user "
        f"(independent inspector required):\n"
        + "\n".join(violations)
    )

## Notes for the registry

- **Second most-cited standard after HazCom.** Common violations: machine-specific procedures not written for all covered equipment; annual inspections not documented; authorized employees not re-trained when procedures change.
- **Group LOTO (c)(3)(ii):** Complex lockout involving multiple energy sources or multiple employees requires a specific group lockout procedure. Ensure the test for procedure coverage accounts for group-lockout scenarios.
- **Tagout vs. Lockout:** If an energy isolating device is not lockable, tagout may be used — but only with supplemental measures (e.g., removing an isolating circuit element). This adds a PARAMETERIZED condition for any equipment using tagout-only. A supplemental test is recommended.
- **Hot-tap exception:** 1910.147(a)(2)(ii) exempts hot-tap operations under specific conditions. If the facility performs hot-taps, ensure the exception criteria are tested rather than excluded.
