## Scope summary

1910.120 applies to five work categories: (1) cleanup operations at uncontrolled hazardous waste sites, (2) corrective actions at RCRA TSD facilities, (3) voluntary cleanup at sites recognized as uncontrolled, (4) operations involving hazardous waste at TSD facilities, and (5) emergency response to hazardous substance releases. Requirements vary by scope — this file focuses on the most broadly applicable elements: the Safety and Health Program, training, medical surveillance, and emergency response.

"""
1910.120(h) — HAZWOPER Training
Confidence: MEDIUM | Human Review: ASSUMPTION REQUIRED (ASSUME-120-002)
"""
import pytest
from datetime import datetime, timedelta

ROLE_INITIAL_HOURS = {
    "general_site_worker": 40,
    "occasional_visitor": 24,
    "supervisor": 40,
    "emergency_first_responder_awareness": 0,  # "sufficient to demonstrate competency"
    "first_responder_operations": 8,
    "hazmat_technician": 24,
    "hazmat_specialist": 24,
    "incident_commander": 24,
}

ANNUAL_REFRESHER_HOURS = 8
REFRESHER_INTERVAL_DAYS = 365


@pytest.mark.assumption(
    id="ASSUME-120-002",
    description="8-hour annual refresher adequate for stable-site workers covering regulatory updates, incidents, HASP",
    approved_by="Compliance Officer",
    review_date="2026-05-20",
)
def test_hazwoper_initial_training_hours_met(employees, training_records):
    violations = []
    for emp in employees:
        role = emp.get("hazwoper_role")
        if not role:
            continue
        min_hours = ROLE_INITIAL_HOURS.get(role)
        if min_hours is None:
            continue
        records = [
            r for r in training_records
            if r["employee_id"] == emp["employee_id"]
            and r["standard_id"] == "1910.120"
            and r.get("training_type") == "initial"
        ]
        if not records:
            violations.append(
                f"{emp['employee_id']} ({role}): no initial HAZWOPER training"
            )
            continue
        total_hours = sum(r.get("hours", 0) for r in records)
        if total_hours < min_hours:
            violations.append(
                f"{emp['employee_id']} ({role}): {total_hours}h initial training "
                f"< required {min_hours}h"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} employee(s) below initial HAZWOPER training hours:\n"
        + "\n".join(violations)
    )


@pytest.mark.assumption(id="ASSUME-120-002")
def test_hazwoper_annual_refresher_current(employees, training_records):
    violations = []
    for emp in employees:
        if not emp.get("hazwoper_role"):
            continue
        refreshers = [
            r for r in training_records
            if r["employee_id"] == emp["employee_id"]
            and r["standard_id"] == "1910.120"
            and r.get("training_type") == "refresher"
        ]
        if not refreshers:
            violations.append(
                f"{emp['employee_id']}: no HAZWOPER annual refresher on record"
            )
            continue
        most_recent = max(refreshers, key=lambda r: r["training_date"])
        age = datetime.utcnow().date() - most_recent["training_date"]
        if age.days > REFRESHER_INTERVAL_DAYS:
            violations.append(
                f"{emp['employee_id']}: refresher {age.days} days ago "
                f"(threshold {REFRESHER_INTERVAL_DAYS})"
            )
        if most_recent.get("hours", 0) < ANNUAL_REFRESHER_HOURS:
            violations.append(
                f"{emp['employee_id']}: refresher was {most_recent.get('hours')}h "
                f"< required {ANNUAL_REFRESHER_HOURS}h"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} employee(s) with expired/insufficient "
        f"HAZWOPER refresher:\n" + "\n".join(violations)
    )

## Notes for the registry

- **Decontamination (k):** Decontamination procedures are explicitly required and must be site-specific. A test verifying the HASP contains decontamination procedures for each contamination type on site is recommended.
- **Air monitoring (h)(8)(i) and (i)):** Air monitoring is required at the beginning of work and during tasks that may expose employees. Monitoring records must link to the HASP. A test verifying monitoring records exist for each site entry day is recommended for high-risk operations.
- **Emergency response vs. incidental release:** 1910.120(q) applies to emergency response operations. If a facility determines that a hazardous substance release can be handled as an "incidental release" (no specialized PPE, evacuation not required, no risk of escalation), it is excluded from (q) but the determination must be documented.
- **1926.65:** Construction-specific HAZWOPER requirements are at 29 CFR 1926.65. They mirror 1910.120 but are technically separate. If construction workers perform HAZWOPER operations, both standards may apply simultaneously.
