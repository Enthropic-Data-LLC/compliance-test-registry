## Scope summary

1910.38 applies to any employer required by an OSHA standard to have an Emergency Action Plan (EAP). It also applies when an employer chooses to rely on evacuation as the primary response to a fire or other emergency (rather than fighting fires per 1910.157). The standard is prescriptive: it enumerates required plan elements, alarm requirements, evacuation roles, and training obligations.

---

## (c) — Elements of the Emergency Action Plan

### Source excerpt

> *An employer must have a written EAP. The plan must include the following elements: (1) Procedures for reporting a fire or other emergency; (2) Procedures for emergency evacuation, including type of evacuation and exit route assignments; (3) Procedures for employees who remain to operate critical plant operations before they evacuate; (4) Procedures to account for all employees after evacuation; (5) Procedures for employees performing rescue or medical duties; (6) The name or job title of every employee who may be contacted by employees who need more information about the plan or an explanation of their duties under the plan.*

```python
"""
1910.38(d) — Employee Alarm System
Confidence: HIGH | Human Review: NOT REQUIRED
"""
from datetime import datetime, timedelta

ALARM_TEST_INTERVAL_DAYS = 180  # 1910.165 requires testing every 2 months


def test_alarm_system_has_distinct_signals(alarm_systems):
    for system in alarm_systems:
        assert system.get("distinct_signals_mapped"), (
            f"VIOLATION: Alarm system at {system.get('site_id')} does not have "
            f"distinct signals mapped per emergency type."
        )


def test_alarm_system_tested_within_interval(alarm_systems):
    violations = []
    for system in alarm_systems:
        last_test = system.get("last_tested_date")
        if not last_test:
            violations.append(
                f"{system.get('site_id')}: no alarm test date on record"
            )
            continue
        age = datetime.utcnow().date() - last_test
        if age.days > ALARM_TEST_INTERVAL_DAYS:
            violations.append(
                f"{system.get('site_id')}: alarm last tested {age.days} days ago "
                f"(threshold {ALARM_TEST_INTERVAL_DAYS})"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} alarm system(s) overdue for testing:\n"
        + "\n".join(violations)
    )
```

```python
"""
1910.38(f) — EAP Employee Training
Confidence: HIGH | Human Review: NOT REQUIRED
"""


def test_all_employees_trained_on_eap(employees, training_records):
    violations = []
    for emp in employees:
        records = [
            r for r in training_records
            if r["employee_id"] == emp["employee_id"]
            and r["standard_id"] == "1910.38"
        ]
        if not records:
            violations.append(
                f"{emp['employee_id']} ({emp['job_title']}): no EAP training on record"
            )
            continue
        earliest = min(r["training_date"] for r in records)
        if earliest > emp.get("start_date", earliest):
            violations.append(
                f"{emp['employee_id']}: EAP training ({earliest}) after "
                f"hire date ({emp['start_date']})"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} employee(s) lack EAP training:\n"
        + "\n".join(violations)
    )
```

## Notes for the registry

- **Small employer exception:** Employers with 10 or fewer employees may communicate the EAP orally rather than in writing. This should be a field in the `eap_document` schema (`is_written` may be `false` for employers with `employee_count <= 10` — the test should check the count before asserting written form).
- **1910.165 cross-reference:** The alarm system test interval is set by 1910.165(d)(2): tested every 2 months if not self-testing. Include a reference to the 1910.165 alarm maintenance tests from this test file.
- **EAP vs. Fire Prevention Plan:** If the employer also maintains a Fire Prevention Plan (1910.39), they may combine it with the EAP in a single document. The combined test should verify both standards' required elements.
