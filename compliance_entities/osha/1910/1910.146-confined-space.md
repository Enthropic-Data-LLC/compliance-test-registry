## Scope summary

1910.146 applies to general industry workplaces that contain permit-required confined spaces (PRCS). A confined space meets three criteria: (1) large enough to enter and work, (2) limited or restricted means of entry or exit, (3) not designed for continuous employee occupancy. It becomes *permit-required* if it contains or has the potential to contain a serious safety or health hazard, contain a hazardous atmosphere, contain material with potential for engulfment, contain an internal configuration that could trap/asphyxiate, or contain any other recognized serious safety hazard.

### `tests/1910.146/test_permit_completeness.py`

```python
"""
1910.146(c)(5)-(8), (d) — Entry Permit Completeness
Confidence: HIGH | Human Review: NOT REQUIRED
"""

REQUIRED_PERMIT_FIELDS = {
    "space_id",
    "purpose_of_entry",
    "authorized_entrants",
    "attendants",
    "entry_supervisor",
    "known_hazards",
    "control_measures",
    "atmospheric_test_results",
    "rescue_emergency_services",
    "communication_procedures",
    "required_equipment",
    "permit_duration",
    "issued_date",
    "authorized_by",
}


def test_all_entries_have_permit(entry_log, entry_permits):
    permit_entry_ids = {p["entry_id"] for p in entry_permits}
    no_permit = [e for e in entry_log if e["entry_id"] not in permit_entry_ids]
    assert not no_permit, (
        f"VIOLATION: {len(no_permit)} PRCS entry/entries recorded without an "
        f"issued permit: {[e['entry_id'] for e in no_permit]}"
    )


def test_permits_have_all_required_fields(entry_permits):
    violations = []
    for permit in entry_permits:
        present = {k for k, v in permit.items() if v is not None and v != ""}
        missing = REQUIRED_PERMIT_FIELDS - present
        if missing:
            violations.append(
                f"Permit {permit.get('permit_id', '?')} "
                f"(space {permit.get('space_id', '?')}): missing {missing}"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} entry permit(s) missing required fields:\n"
        + "\n".join(violations)
    )


def test_permit_issued_before_entry(entry_log, entry_permits):
    violations = []
    for entry in entry_log:
        permit = next(
            (p for p in entry_permits if p["entry_id"] == entry["entry_id"]), None
        )
        if permit and permit.get("issued_date") > entry.get("entry_start_time"):
            violations.append(
                f"Entry {entry['entry_id']}: permit issued "
                f"{permit['issued_date']} AFTER entry began "
                f"{entry['entry_start_time']}"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} entry permit(s) issued after entry began:\n"
        + "\n".join(violations)
    )
```

## Notes for the registry

- **Atmospheric testing sequence:** O2 first, then flammables, then toxics — this order is mandated and should be captured in the permit's atmospheric test result sequence field.
- **Non-entry rescue preference:** OSHA prefers non-entry rescue where feasible. If an employer selects entry rescue, additional evidence of adequacy is required. This distinction should be a flag in the rescue service evaluation record.
- **Cancelled permits:** Cancelled permits must be retained for 1 year and be made available for review. Add a test for permit retention compliance.
- **1926 Subpart AA:** The 2015 construction confined space rule has a separate structure with some different definitions (e.g., "controlling contractor"). Construction PRCS entries are not covered by this file.
