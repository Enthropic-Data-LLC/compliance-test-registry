## Scope summary

1926.451 establishes general scaffold requirements for construction. It covers capacity, platform construction, access, use, fall protection, and falling-object protection. Specific scaffold types (supported, suspended, aerial lifts) have additional requirements in 1926.452–454. The competent-person concept is central — most inspection and design determinations require a designated competent person.

The standard applies to all construction scaffolding. The 10-foot fall protection trigger (vs. 6 feet for general fall protection) applies specifically to scaffold workers; this distinction is explicitly noted.

## (b)(1) — Platform Construction: Width and Planking

### Source excerpt

> *Each platform on all working levels of scaffolds shall be fully planked or decked between the front uprights and the guardrail supports.*

> *The space between adjacent planks, and between planks and the platform edge, shall not exceed 1 inch.*

| Element | Value | Classification |

**Assumption (ASSUME-451-001):** The 1-inch gap requirement applies to standard planked platforms. OSHA allows alternative decking materials (fabricated deck units, metal decks) with manufacturer-certified maximum opening sizes. When alternative decking is used, the manufacturer specification — not the 1-inch rule — is the applicable test criterion.

### Source excerpt

```python
"""
1926.451(a) — Scaffold Capacity (4x Load)
Confidence: HIGH | Human Review: NOT REQUIRED
"""

LOAD_FACTOR = 4


def test_scaffold_rated_for_4x_intended_load(scaffold_records):
    violations = []
    for scaffold in scaffold_records:
        rated = scaffold.get("rated_capacity_lbs", 0)
        intended = scaffold.get("intended_load_lbs", 0)
        if intended and rated < intended * LOAD_FACTOR:
            violations.append(
                f"Scaffold {scaffold['scaffold_id']}: rated {rated} lbs, "
                f"intended load {intended} lbs — required {intended * LOAD_FACTOR} lbs "
                f"({LOAD_FACTOR}x minimum)"
            )
    assert not violations, (
        f"CRITICAL VIOLATION: {len(violations)} scaffold(s) undersized for intended load:\n"
        + "\n".join(violations)
    )


def test_scaffold_load_posting_present(scaffold_records):
    violations = [
        s for s in scaffold_records
        if not s.get("load_posting_present")
    ]
    assert not violations, (
        f"VIOLATION: {len(violations)} scaffold(s) missing load capacity posting: "
        f"{[s['scaffold_id'] for s in violations]}"
```

```python
"""
1926.451(b)(1) — Platform Planking and Gaps
Confidence: MEDIUM | Human Review: ASSUMPTION REQUIRED (ASSUME-451-001)
"""
import pytest

MAX_GAP_INCHES = 1.0


@pytest.mark.assumption(
    id="ASSUME-451-001",
    description=(
        "Alternative decking: manufacturer spec governs gap size, not the 1-inch rule. "
        "Document exception_applicable and manufacturer_spec_max_opening."
    ),
    approved_by="Compliance Officer",
    review_date="2026-05-20",
)
def test_platform_gap_within_tolerance(platform_inspections):
    violations = []
    for inspection in platform_inspections:
        if inspection.get("exception_applicable"):
            allowed_gap = inspection.get("manufacturer_spec_max_opening_inches", MAX_GAP_INCHES)
        else:
            allowed_gap = MAX_GAP_INCHES
        actual_gap = inspection.get("max_gap_inches", 0)
        if actual_gap > allowed_gap:
            violations.append(
                f"Scaffold {inspection['scaffold_id']}: platform gap {actual_gap}\" "
                f"exceeds allowed {allowed_gap}\" "
                f"({'exception applied' if inspection.get('exception_applicable') else 'standard 1-inch rule'})"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} scaffold platform(s) with excessive gaps:\n"
        + "\n".join(violations)
    )


def test_platform_fully_planked(platform_inspections):
    violations = [
        p for p in platform_inspections
        if not p.get("fully_planked") and not p.get("exception_applicable")
    ]
    assert not violations, (
        f"VIOLATION: {len(violations)} scaffold platform(s) not fully planked: "
        f"{[p['scaffold_id'] for p in violations]}"
    )
```

## Notes for the registry

- **Competent person for scaffolds:** The competent person for scaffolding must be qualified specifically in scaffold hazards (not just general construction). Their designation should be documented per scaffold type used on site.
- **Scaffold inspection:** Scaffolds must be inspected before each work shift by a competent person and after any occurrence that could affect structural integrity (e.g., storm, impact). Inspection records should be maintained per shift.
- **Aerial lifts (1926.453):** Aerial work platforms are governed by 1926.453, not 1926.451. However, 1926.451 fall protection provisions apply to all scaffolding — aerial lifts must use harnesses. Tests for aerial lift use should reference 1926.453 rather than this file.
- **Scaffold tagging:** Many employers use a tag system (green = safe to use, red = do not use) on scaffolds. While not required by OSHA, it is a best practice that supports the inspection documentation requirement.
