## Scope summary

Subpart P (1926.650–652) governs excavation and trenching operations. 1926.651 covers specific requirements during excavation (access/egress, underground utilities, atmospheric hazards, inspections). 1926.652 covers protective systems (sloping, shoring, trench boxes). The competent person concept is foundational — they must classify soil, select protective systems, and conduct daily/event-triggered inspections.

Excavations > 5 feet deep (or less if unstable soil) require a protective system or sloping. Excavations ≥ 20 feet deep require a protective system designed by a Registered Professional Engineer.

## 1926.651(g) — Hazardous Atmospheres

### Source excerpt

> *In addition to the requirements set forth in subparts D and E of this part, protection from hazardous atmospheres shall include ... where oxygen deficiency or a hazardous atmosphere exists or could reasonably be expected to exist, the atmosphere shall be tested before employees enter the excavation.*

> *Excavations shall be inspected after every rain storm or other hazard-increasing occurrence, and at the start of each shift.*

### Element extraction

| Element | Value | Classification |
|---|---|---|
| Subject | Excavations where hazardous atmosphere is possible | PARAMETERIZED |
| Condition | Before employee entry AND after rain/event AND at start of each shift | DETERMINISTIC |
| Obligation | Atmosphere tested for O2 (acceptable: 19.5%–23.5%), flammables (<10% LEL), CO, H2S as applicable; results documented | PARAMETERIZED |
| Evidence | `atmospheric_test_record.excavation_id`, `test_date`, `o2_pct`, `lel_pct`, `co_ppm`, `h2s_ppm`, `result_in_range == true` | PARAMETERIZED |

**Assumption (ASSUME-651-001):** Atmospheric testing is required whenever: (1) excavation is in or near a landfill, waste site, or industrial area; (2) excavation intersects or is near underground utility infrastructure; (3) signs of off-gas are observed; or (4) excavation is deeper than 20 feet. "Could reasonably be expected" is interpreted as any of these four conditions. Excavations in undeveloped agricultural land with verified absence of underground utilities do not require atmospheric testing under this assumption.

**Overall: PARAMETERIZED → Pattern 2**

## 1926.652 — Protective Systems

### Source excerpt

> *Each employee in an excavation shall be protected from cave-ins by an adequate protective system.*

| Depth | Soil Type | Required System |
|---|---|---|
| < 5 feet | Any | May be exempt if no hazard indicators (competent person determination) |
| ≥ 5 feet | Type C (weakest) | Slope 1.5:1 (H:V) or shoring/trench box |
| ≥ 5 feet | Type B | Slope 1:1 or shoring/trench box |
| ≥ 5 feet | Type A (strongest) | Slope 0.75:1 or shoring/trench box |
| ≥ 20 feet | Any | Must be designed by Registered Professional Engineer |

### Element extraction

| Element | Value | Classification |
|---|---|---|
| Subject | Each employee in an excavation ≥ 5 feet deep | DETERMINISTIC |
| Condition | Employee is working in the excavation | DETERMINISTIC |
| Obligation | Protective system installed per soil classification; system matches OSHA Appendix B, C, or D tables; excavations ≥ 20 feet have PE-designed system | DETERMINISTIC |
| Evidence | `protective_system_record.excavation_id`, `depth_feet`, `soil_classification`, `system_type`, `meets_osha_table == true`, `pe_designed` where `depth_feet >= 20` | DETERMINISTIC |

**Overall: DETERMINISTIC → Full Automation (Pattern 1)** for the basic protective-system requirement (soil classification and table lookup). Soil classification itself requires competent person judgment — see Pattern 2 note below.

**Note on soil classification:** Soil classification (A, B, C) is performed by the competent person using OSHA Appendix A tests (visual, manual). This determination is PARAMETERIZED — it requires competent person judgment. The test verifies the classification was documented and made by a competent person, not that the classification itself is correct.

```python
"""
1926.652 — Protective Systems (Cave-in Protection)
Confidence: HIGH for system requirement; MEDIUM for classification
"""

SLOPE_REQUIREMENTS = {
    "A": 0.75,   # H:V ratio (max) — 3/4:1
    "B": 1.0,    # H:V ratio (max) — 1:1
    "C": 1.5,    # H:V ratio (max) — 1.5:1
}

PROTECTIVE_SYSTEM_DEPTH_FT = 5
PE_DESIGN_REQUIRED_DEPTH_FT = 20


def test_protective_system_installed_at_5_feet(excavation_records, protective_system_records):
    ps_excavation_ids = {ps["excavation_id"] for ps in protective_system_records}
    violations = [
        r for r in excavation_records
        if r.get("depth_feet", 0) >= PROTECTIVE_SYSTEM_DEPTH_FT
        and r["excavation_id"] not in ps_excavation_ids
    ]
    assert not violations, (
        f"CRITICAL VIOLATION: {len(violations)} excavation(s) ≥{PROTECTIVE_SYSTEM_DEPTH_FT}ft "
        f"deep with no protective system: {[r['excavation_id'] for r in violations]}"
    )


def test_pe_design_required_at_20_feet(protective_system_records):
    violations = [
        ps for ps in protective_system_records
        if ps.get("excavation_depth_ft", 0) >= PE_DESIGN_REQUIRED_DEPTH_FT
        and not ps.get("pe_designed")
    ]
    assert not violations, (
        f"CRITICAL VIOLATION: {len(violations)} excavation(s) ≥{PE_DESIGN_REQUIRED_DEPTH_FT}ft "
        f"deep without PE-designed protective system: "
        f"{[ps['excavation_id'] for ps in violations]}"
    )


def test_slope_ratio_meets_soil_classification(protective_system_records):
    violations = []
    for ps in protective_system_records:
        if ps.get("system_type") != "sloping":
            continue
        soil = ps.get("soil_classification")
        max_ratio = SLOPE_REQUIREMENTS.get(soil)
        if max_ratio is None:
            continue
        actual_ratio = ps.get("actual_slope_h_v_ratio", 0)
        if actual_ratio > max_ratio:
            violations.append(
                f"Excavation {ps['excavation_id']}: soil {soil}, "
                f"slope {actual_ratio}:1 exceeds max {max_ratio}:1"
            )
    assert not violations, (
        f"VIOLATION: {len(violations)} excavation slope(s) exceed the maximum "
        f"allowed for the soil classification:\n" + "\n".join(violations)
    )
```

## Notes for the registry

- **Soil classification is the competent person's determination.** The test verifies it was documented and made by a designated competent person — it cannot verify the classification is correct. Physical testing (manual test, visual indicators) must be recorded in the classification record to make it defensible.
- **Layered soils:** When multiple soil types are present at different depths, the most conservative classification governs unless each layer is analyzed separately and protective systems transition at the appropriate depth. This adds complexity to the slope ratio test.
- **Spoils:** Spoil (excavated material) must be placed at least 2 feet from the edge of the excavation. Add a test: `spoil_placement_record.distance_from_edge_ft >= 2`.
- **Water accumulation:** 1926.651(h) requires that employees be protected from water accumulation. Pumping and dewatering equipment must be inspected. Add a test verifying dewatering is addressed in the site safety plan when ground water is possible.
- **Crossing over excavations:** Mobile equipment must have barricades or stop logs. Bridges over excavations must be designed by a qualified person. These add DETERMINISTIC tests if bridging/crossing records exist in the SMS.
